# Генератор новогодней ёлки
Простой инструмент для создания изображений новогодних ёлок в текстовом формате.
# Возможности
 - Генерация ёлок разной высоты (от 2х этажей и выше).
 - Автоматическое центрирование всех элементов.
 - Вывод в текстовый файл.
 - Валидация входных данных.
 - Покрытие тестами.
# Использование
Запустите скрипт с двумя аргументами:
node generateTree.js <количество_этажей> <путь_к_файлу>
# Формат вывода
``` 
                 W
                 *
            @* * * * * 
        * * * * * * * * * @
    @* * * * * * * * * * * * * 
* * * * * * * * * * * * * * * * * @
               TTTTT
               TTTTT
```
